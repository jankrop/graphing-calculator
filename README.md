## How to run

You will find a .apk file with the application in the Releases.

If you insist on building from source (like the open source freedom fighter you are), the best way to do it is with Android Studio.

- To build an APK: Build > Generate App Bundles or APKs > Generate APKs
- To emulate it on your machine: Run (Shift+F10)

## Technologies used

The standard stack for Android applications, using Kotlin and Gradle.
